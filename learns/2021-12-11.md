### pull

[hub.docker.com](https://hub.docker.com/) > Explore 에서 사용하고 싶은 image 를 찾을 수 있음

```bash
docker pull nginx
```

### run

```
docker run nginx
```

특정 포트 지정하려면..

```bash
docker run -p 8080:80 nginx
```

위 커맨드 실행 후 `localhost:8080` 접속하면 nginx 가 실행되었음을 확인할 수 있다.
